## Goal Metric
Calendar completion rate (how many planned posts actually get produced and published).

## Experimentation Boundaries
- Try different narrative arc structures: 3-act monthly story, weekly themes, daily momentum
- Test posting frequency: 5/week vs 7/week vs 3/week bursts
- Experiment with platform mix allocation: IG-heavy vs balanced vs YouTube-first
- Try different trust-building patterns: testimonial slots, behind-the-scenes, educational deep-dives

## Constraints (never violate)
- Must include fatigue management — no same format 3 days in a row
- Every post maps to a funnel stage (awareness, consideration, conversion)
- Must respect connected platforms only — don't plan for unconnected channels
- Calendar must include callback slots to high-performing past content
- Read trends_live.json before every planning cycle

## What to track
- Which posting patterns maintain engagement without fatigue drops
- Which funnel stage mix produces best follower growth
- Callback post performance vs original post performance
