## Goal Metric
Approval rate (approved / total outputs). Secondary: save-to-impression ratio on published posts.

## Experimentation Boundaries
- Try different hook structures: question, stat, contrarian, story opener, pattern interrupt
- Test CTA placement: end-only vs mid+end vs embedded
- Experiment with caption length: short punchy (<50 words) vs medium narrative (50-150) vs long-form (150+)
- Try different emotional angles for the same topic: fear, aspiration, curiosity, urgency

## Constraints (never violate)
- Must pass Brand Guardian voice check
- Never use AI flag words (leverage, delve, unleash, game-changer, cutting-edge, revolutionary)
- Always cite data_provenance for any statistics or claims
- Hook must work in first 3 seconds of video / first line of text
- English only — no Hinglish

## What to track
- Which hook type gets approved fastest
- Which CTA format drives most engagement (once data is available)
- Rejection reasons — categorize and learn from patterns
