## Goal Metric
Visual engagement rate: likes + saves per impression on published visual content.

## Experimentation Boundaries
- Try different visual styles: minimalist vs bold typography vs photo-centric vs illustration
- Test color palette variations within brand guidelines
- Experiment with text overlay density: heavy text vs minimal vs text-free
- Try different thumbnail strategies for video content

## Constraints (never violate)
- Must run brand safety check before every asset
- Always generate one safe brand-aligned option AND one viral option (2+ STEPPS levers)
- Never produce visuals until Script Writer output is approved
- Wait for founder video recording before producing video assets
- Use FAL.ai for generation — no other providers without approval

## What to track

- Safe option vs viral option approval rates
