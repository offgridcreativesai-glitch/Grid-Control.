## Goal Metric
Actionability score: can the content team execute this strategy within the given timeframe without additional research?

## Experimentation Boundaries
- Try different competitor analysis frameworks: SWOT, Jobs-to-be-done, blue ocean, positioning maps
- Test different roadmap structures: theme-based quarters vs metric-based sprints vs event-driven
- Experiment with depth vs breadth: deep-dive on 3 competitors vs surface scan of 10
- Try contrarian positioning vs market-following vs category-creating angles

## Constraints (never violate)
- Every claim must cite a real data source (Apify scrape, API call, or verified public data)
- Never fabricate competitor metrics
- Always include feasibility assessment with resource requirements
- decision_engine: pure_math where applicable — no Claude for final strategic decisions

## What to track
- Which framework produces most actionable recommendations
- How often strategic recommendations get implemented vs shelved
- Time from strategy output to first content piece executing that strategy
